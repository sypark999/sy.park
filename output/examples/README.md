## 🎨 디자인 특징

### 레이아웃
- **상단 (1080x675)**: 메인 음식 이미지 - 가장 눈에 띄는 배치
- **하단 좌측 (540x675)**: 서브 이미지 1 - 추가 메뉴 또는 다른 각도
- **하단 우측 (540x675)**: 서브 이미지 2 - 보조 이미지

### 텍스트
- **식당명**: 80px, 흰색 + 검정 테두리 (굵게)
- **주소**: 40px, 흰색 + 검정 테두리
- **위치**: 중앙 상단, 이미지 위에 오버레이

### 색상
- 텍스트: `#FFFFFF` (흰색)
- 테두리: `#000000` (검정)
- 테두리 두께: 식당명 8px, 주소 6px

## 📱 Instagram 업로드 가이드

### 권장 사용처
- ✅ Instagram 스토리
- ✅ Instagram 릴스
- ✅ Instagram 피드 (4:5 비율)
- ✅ Facebook 스토리

### 업로드 방법
1. 이미지를 휴대폰으로 전송 (AirDrop, 메신저 등)
2. Instagram 앱 열기
3. 새 게시물 또는 스토리 선택
4. 이미지 업로드
5. 필요시 필터/효과 추가
6. 해시태그 및 위치 태그 추가
7. 게시!

### 추천 해시태그 (중국어)
```
#首尔美食 #弘大美食 #韩国料理
#首尔旅游 #弘大 #韩国旅行
#美食推荐 #餐厅推荐
```
