# 레스토랑 템플릿 사용 가이드

## 새로운 기능! 🎉

레퍼런스 이미지를 바탕으로 **3개 이미지 합성 레스토랑 템플릿**이 추가되었습니다!

## 📐 레이아웃 구조

```
1080 x 1350 (Instagram Portrait)
┌─────────────────────────┐
│                         │
│    메인 이미지           │ 1080x675
│    (상단 전체)           │
│                         │
├───────────┬─────────────┤
│           │             │
│  서브 1    │   서브 2     │ 각 540x675
│  (하단좌)  │   (하단우)   │
│           │             │
└───────────┴─────────────┘
```

**텍스트**:
- 중앙 상단: 식당명 (큰 글씨, 흰색 + 검정 테두리)
- 하단: 주소 (작은 글씨, 흰색 + 검정 테두리)

## 🚀 사용 방법

### 1. 워크플로우 실행
```bash
python3 sns_workflow.py
```

### 2. 템플릿 선택
```
템플릿을 선택하세요:
1. Instagram Portrait
2. Instagram Square
3. Instagram Restaurant (3 Images)  ← 이거!
```

### 3. 이미지 3개 선택
```
[1/3] 메인 이미지 (상단 전체)
→ 가장 눈에 띄는 대표 음식 이미지

[2/3] 서브 이미지 1 (하단 좌측)
→ 보조 이미지 (다른 메뉴 or 매장 외관)

[3/3] 서브 이미지 2 (하단 우측)
→ 보조 이미지 (또 다른 메뉴 or 분위기)
```

### 4. 텍스트 입력
```
식당명을 입력하세요: 옥동식
주소를 입력하세요: 서울 마포구 홍대입구
```

### 5. 결과 확인
미리보기가 `output/preview/` 폴더에 생성됩니다.
만족하면 저장, 아니면 다시 만들기!

## 🎨 디자인 특징

### 텍스트 스타일
- **흰색 글씨 + 검정 테두리**: 어떤 배경에서도 잘 보임
- **중앙 정렬**: 깔끔하고 균형잡힌 레이아웃
- **크기 계층**: 식당명(80px) > 주소(40px)

### 이미지 배치
- 상단 메인: 전체 너비 활용, 임팩트 있는 비주얼
- 하단 2개: 추가 정보 전달, 다양성 표현

## 📊 비교표

| 템플릿 | 크기 | 이미지 개수 | 용도 |
|--------|------|------------|------|
| Instagram Square | 1080x1080 | 1개 | 피드 게시물 |
| Instagram Portrait | 1080x1350 | 1개 | 스토리/릴스 |
| **Instagram Restaurant** | **1080x1350** | **3개** | **레스토랑 홍보** |

## 🎯 활용 팁

### 이미지 선택 전략

**메인 이미지 (상단)**:
- 가장 시그니처 메뉴
- 색감이 화려하고 식욕을 돋우는 음식
- 플레이팅이 멋진 요리

**서브 이미지 1 (좌하)**:
- 다른 인기 메뉴
- 매장 외관
- 내부 인테리어

**서브 이미지 2 (우하)**:
- 또 다른 메뉴
- 사이드 메뉴
- 분위기 샷

### 텍스트 작성 팁

**식당명**:
- 정확한 상호명
- 한글 + 영문 가능
- 예: "옥동식", "Okdongsik", "옥동식 OKDONGSIK"

**주소**:
- 간결하게 (구/동 정도)
- 예: "서울 마포구 홍대입구"
- 또는 특징: "홍대입구역 5번 출구 앞"
- 빈칸으로 두면 표시 안됨

## 🔧 커스터마이징

### 텍스트 위치 조정
`templates/instagram_restaurant.json` 수정:

```json
{
  "type": "text",
  "name": "restaurant_name",
  "position": [540, 450],  ← [x, y] 좌표
  "font_size": 80,         ← 크기
  "stroke_width": 8        ← 테두리 두께
}
```

### 색상 변경
```json
{
  "color": "#FFFFFF",         // 텍스트: 흰색
  "stroke_color": "#000000"   // 테두리: 검정
}
```

### 이미지 영역 크기 조정
```json
{
  "type": "main_image",
  "size": [1080, 675]  ← [너비, 높이]
}
```

## 🆚 기존 템플릿과 차이점

### 기존 (Portrait/Square)
- 1개 이미지만 사용
- 그라디언트 오버레이
- 제목 + 부제목

### 새로운 (Restaurant)
- **3개 이미지 합성** ✨
- 배경 오버레이 없음 (이미지 자체가 배경)
- 텍스트 테두리로 가독성 확보
- 식당명 + 주소

## ❓ 문제 해결

### Q: 텍스트가 안 보여요
A: 테두리가 있어서 어떤 배경에서도 보여야 합니다. `stroke_width`를 더 늘려보세요.

### Q: 이미지가 이상하게 잘려요
A: 이미지가 자동으로 중앙 크롭됩니다. 원본 이미지의 주요 피사체가 중앙에 있는지 확인하세요.

### Q: 3개 이미지가 너무 번거로워요
A: 기존 Portrait/Square 템플릿을 사용하세요! (1개 이미지만 필요)

### Q: 같은 이미지를 3번 선택해도 되나요?
A: 네! 하지만 3개 다른 이미지를 쓰는 게 더 효과적입니다.
